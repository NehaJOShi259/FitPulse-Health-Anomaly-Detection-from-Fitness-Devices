Objective: Clean and consolidate fitness datasets to 1-minute intervals.

Dataset source: dailyActivity_merged.csv, heartrate_seconds_sample.csv, sleepDay_merged.csv.

Steps performed:

1.Load CSVs

2.Convert timestamps to datetime

3.Remove duplicates

4.Resample datasets to 1-minute intervals

5.Merge datasets into a final clean dataset

Tools used: Pandas, NumPy, Matplotlib, Seaborn (optional)

Key outputs: Screenshots of loaded data (data_loaded.png) and merged dataset (merged_head().png).
